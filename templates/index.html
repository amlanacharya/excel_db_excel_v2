                 About This System
            

            

                
This Excel Natural Language Query System allows you to ask questions about your Excel data in plain English. The system uses:

                
                
    
                    
  • Natural Language Processing to understand your questions
    • 
                    
  • Retrieval-Augmented Generation (RAG) to find relevant context
    • 
                    
  • SQL Generation to create database queries
    • 
                    
  • Your Excel Data stored in SQLite for quick access
    • 
                

                
                
You can ask questions like:

                
    
                    
  • "What was the total amount of loans in June?"
    • 
                    
  • "Show me deposits greater than $3000"
    • 
                    
  • "Calculate the average interest rate for loans by month"
    • 
                    
  • "Compare deposits and loans by category"
    • 
                

                
                
For best results:

                
    
                    
  • Be specific about what data you're looking for
    • 
                    
  • Mention column names when possible
    • 
                    
  • Ask one question at a time
    •
